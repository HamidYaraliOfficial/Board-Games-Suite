# 🎮 Board Games Suite

---

# 🇺🇸 English

## Overview

Board Games Suite is a modern desktop application developed with Python and PyQt6 that includes two classic board games in a single interface:

- ♟️ Chess
- ⚫⚪ Othello (Reversi)

The application features a modern graphical interface, multilingual support, theme switching, AI opponents, move validation, undo functionality, game hints, and responsive board rendering.

---

## Features

### Chess

- Complete chess rules implementation
- Castling support
- En Passant support
- Pawn Promotion
- Check Detection
- Checkmate Detection
- Stalemate Detection
- Undo Moves
- Move Hints
- Human vs Human
- Human vs AI

### Othello

- Complete Othello game mechanics
- Automatic piece flipping
- Score tracking
- Pass-turn handling
- Winner detection
- Undo support
- Human vs Human
- Human vs AI

### General

- Dark Theme
- Light Theme
- Multilingual Interface
  - English
  - Persian (فارسی)
  - Chinese (中文)
- Responsive UI
- Modern PyQt6 Design

# 🇨🇳 中文

## 项目简介

Board Games Suite 是一个使用 Python 和 PyQt6 开发的现代桌面应用程序，集成了两款经典棋盘游戏：

- ♟️ 国际象棋（Chess）
- ⚫⚪ 黑白棋（Othello / Reversi）

该项目提供现代化图形界面、多语言支持、主题切换、AI 对战、悔棋功能以及完整游戏规则实现。

---

## 功能特点

### 国际象棋

- 完整国际象棋规则
- 王车易位（Castling）
- 吃过路兵（En Passant）
- 兵升变
- 将军检测
- 将死检测
- 逼和检测
- 悔棋功能
- 走法提示
- 双人模式
- AI 对战模式

### 黑白棋

- 完整黑白棋规则
- 自动翻转棋子
- 实时计分
- 自动跳过回合
- 胜负判断
- 悔棋功能
- 双人模式
- AI 对战模式

### 通用功能

- 深色主题
- 浅色主题
- 多语言支持
  - English
  - فارسی
  - 中文
- 响应式界面
- 现代化 PyQt6 设计
